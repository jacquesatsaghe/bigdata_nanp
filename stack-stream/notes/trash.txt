    genre VARCHAR(10) CHECK (genre IN ('M', 'F')),
    val1 INTEGER CHECK (val1 >= 0) NOT NULL,

# CONNECT

** list of existing connectors:
	curl -s -X GET http://localhost:8083/connector-plugins | jq '.[].class'
	
** list of active connectors:
	curl -s http://localhost:8083/connectors | jq
	
** others:
Liste des plugins dispos	GET	/connector-plugins
Liste des instances actives	GET	/connectors
Détails d'un connecteur	GET	/connectors/{name}
Statut d'un connecteur	GET	/connectors/{name}/status
Configuration d'un connecteur	GET	/connectors/{name}/config

** submit connector
curl -X POST http://localhost:8083/connectors -H "Content-Type: application/json" -d @source-postgres-debezium.json

** submit sink
curl -X POST http://localhost:8083/connectors -H "Content-Type: application/json" -d @sink-elastic-debezium.json

** connectors status

# status Pour le connecteur Source
curl -s http://localhost:8083/connectors/postgres-source-connector-dbz/status | jq

# Pour le connecteur Sink
curl -s http://localhost:8083/connectors/elasticsearch-sink-connector-dbz/status | jq

** pause a connector
curl -X PUT http://localhost:8083/connectors/elasticsearch-sink-connector-dbz/pause
curl -X PUT http://localhost:8083/connectors/postgres-source-connector-dbz/pause

** stop a connector
curl -X PUT http://localhost:8083/connectors/elasticsearch-sink-connector-dbz/stop
curl -X PUT http://localhost:8083/connectors/postgres-source-connector-dbz/stop

** delete a connector
curl -X DELETE http://localhost:8083/connectors/elasticsearch-sink-connector-dbz
curl -X DELETE http://localhost:8083/connectors/postgres-source-connector-dbz

** Redemarrage apres FAILED
	** juste le connecteur 
	curl -X POST http://localhost:8083/connectors/elasticsearch-sink-connector-dbz/restart
	curl -X POST http://localhost:8083/connectors/postgres-source-connector-dbz/restart
	
	** juste une tache (ex 0)
	curl -X POST http://localhost:8083/connectors/elasticsearch-sink-connector-dbz/tasks/0/restart
	curl -X POST http://localhost:8083/connectors/postgres-source-connector-dbz/tasks/0/restart
	
	** complet
	curl -X POST http://localhost:8083/connectors/elasticsearch-sink-connector-dbz/restart?includeTasks=true&onlyFailed=false
	curl -X POST http://localhost:8083/connectors/postgres-source-connector-dbz/restart?includeTasks=true&onlyFailed=false
	
** reprendre apres une PAUSE & un STOP (passer à l'etat RUNNING):
curl -X PUT http://localhost:8083/connectors/elasticsearch-sink-connector-dbz/resume
curl -X PUT http://localhost:8083/connectors/postgres-source-connector-dbz/resume

** Reinitialiser et reprendre depuis le debut (3 etapes):
	** 1- arreter le connector (PUT /stop)
	
	** 2- Réinitialisez les offsets
	curl -X DELETE http://localhost:8083/connectors/elasticsearch-sink-connector-dbz/offsets
	curl -X DELETE http://localhost:8083/connectors/postgres-source-connector-dbz/offsets
	
	** 3- Relancez : PUT /connectors/{name}/resume
